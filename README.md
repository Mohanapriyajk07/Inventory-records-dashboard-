📊 Inventory Records Dashboard (Excel Project)

📌 Overview

This project is an Inventory Records Dashboard built using Microsoft Excel. It helps analyze product performance, stock levels, and inventory cost through interactive visualizations and key performance indicators (KPIs).

🎯 Objectives

Track units sold and stock availability

Identify top-selling products

Monitor inventory cost

Present data in a clear and visual dashboard format

🛠️ Tools & Technologies

Microsoft Excel

Pivot Tables

Pivot Charts

Data Cleaning & Formatting

📂 Dataset Details

The dataset contains product-level inventory information, including:

Product ID

Product Name

Opening Stock

Purchase In

Units Sold

Stock in Hand

Cost Price per Unit

Total Inventory Cost

📊 Dashboard Features

📈 Top Selling Products (based on units sold)

📦 Current Stock Levels (available inventory)

💰 Inventory Cost by Product

📌 KPIs:

Total Units Sold

Total Inventory Cost

💡 Key Insights

Identified high-performing products based on sales

Detected low-stock items requiring restocking

Analyzed cost distribution across products
